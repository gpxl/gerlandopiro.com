Work

Some things I've worked on. There's some Rails, Jekyll, Wordpress, Bootstrap, Engine Yard Cloud and Digital Ocean to varying degrees.

Engineyard.com

This was a multi-department rebranding effort that included a complete backend overhaul from Rails to Jekyll. My responsibilities included definition and set up of the backend services, front-end development, and migration of content from the previous Rails CMS.

Distill 2014

A promotional site for Engine Yard's yearly Distill conference. This was built using Jekyll and supported by a small Rails app to process speaker proposals. My responsibilities included set up of the backend services, front-end development, and Rails development.

Leap Year Project

Here's a Wordpress site complimented by a small Rails app to process Leap Night submissions. I handled backend services and development, Johnathon Strube worked the pixels, and Victor Saad provided content and vision.

Color Palettes

This is a little Rails app running I built as a first step in a Sass refactoring tool, but I ran out of steam/interest before going any further. It's ugly and buggy, but it's still interesting to see people from all over the world add sites to the list of submissions.
